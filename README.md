# 藪の中 - オンラインカードゲーム

[Oink Games「藪の中」](https://oinkgames.com/ja/games/analog/in-a-grove/)を元にしたオンラインマルチプレイヤーゲーム。芥川龍之介の小説に着想を得た推理と欺きのカードゲームです。

## 必要環境

- **Docker** & **Docker Compose** (バックエンド)
- **Node.js** v18+ & **npm** (フロントエンド)

## クイックスタート

```sh
# 1. 初回セットアップ
make setup

# 2. 開発環境を起動（サーバー + クライアント）
make dev
```

ブラウザで http://localhost:5173 を開いてプレイ。

## ゲームルール

### 概要

3人の容疑者の中から犯人を推理するゲームです。プレイヤーは証拠を集め、犯人を告発します。間違った告発にはペナルティが課せられ、ペナルティが溜まりすぎたプレイヤーが敗北します。

### ゲームの流れ

1. **アリバイ確認** - 各プレイヤーに配られたアリバイカード（容疑者にいない人物）を確認。3人以上の場合は右隣のプレイヤーのカードも見られる
2. **容疑者確認** - 3人の容疑者のうち2人のカードを確認（発見者から順番に）
3. **すり替え** (発見者のみ) - 確認した容疑者と被害者カードを入れ替えることができる
4. **告発** - 犯人だと思う容疑者を選ぶ
5. **結果発表** - 犯人を公開。間違った容疑者を告発したスタックの最上位プレイヤーがペナルティを受ける

### 犯人の決め方

- 容疑者3人の数字カードのうち、**最大値**が犯人
- ただし **5** が容疑者にいる場合、ルールが**反転**し **最小値**が犯人
- **ブランクカード**は犯人になれない

### プレイヤー人数と設定

| 人数 | カード範囲 | 告発チップ | 終了閾値 |
|------|-----------|-----------|---------|
| 2人 | 4-8 + Blank | 5枚 | 8 |
| 3人 | 3-8 + Blank | 5枚 | 8 |
| 4人 | 2-8 + Blank | 5枚 | 8 |
| 5-6人 | 2-9/10 + Blank | 6枚 | 10 |
| 7-8人 | 2-11/12 + Blank | 7枚 | 12 |
| 9-10人 | 2-13/14 + Blank | 8枚 | 14 |

### ゲーム終了条件

以下のいずれかに該当するプレイヤーが**敗北**:

- チップの合計（告発チップ + ライアーチップ）が**終了閾値**以上になった
- 告発チップが **0** になった

## 遊び方

1. ブラウザで http://localhost:5173 にアクセス
2. 名前を入力し「ルームを作成」
3. 表示された4桁のルームIDを他のプレイヤーに共有
4. 他のプレイヤーは同じURLにアクセスし「ルームに参加」からIDを入力
5. 2人以上揃ったらホストが「ゲーム開始」
